# MyRights Design System

## App Name
MyRights

## Visual Direction
The design should feel trustworthy, calm, accessible, clean, and suitable for adult children helping aging parents.

## Colors
Primary: #1E4E79
Secondary: #EAF4FB
Accent: #2E7D32
Background: #F7F9FB
Text: #263238
Error: #D32F2F
Success: #388E3C
Border: #D9E2EA

## Typography
Font: Assistant, Arial, sans-serif

Heading: 28px / Bold  
Body: 18px / Regular  
Caption / Label: 14px / Regular

## Spacing
Base unit: 8px

## Border Radius
12px

## Buttons
Primary button: dark blue background, white text, rounded corners, large and clear.

Secondary button: white background, blue border, blue text, rounded corners.

## Cards
White background, subtle border, light shadow, rounded corners, generous padding.

## Inputs
Large and clear fields with light gray border, readable text, label above the field.

## Navigation
Simple top navigation with the MyRights logo and links to Home, Questionnaire, Dashboard, and Right Details.

## Accessibility
Use large readable text, strong contrast, clear buttons, and RTL Hebrew layout.
